# A machine with souvenirs for tourists in React

## 1. Instal parcel in global

Yarn:

```
yarn global add parcel-bundler
```

npm:

```
npm install -g parcel-bundler
```

## 2. Run npm

```
npm install
```

## 3. Run project

```
npm start
```
